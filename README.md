# README #

Tool for easily build models based on [allenNLP](http://allennlp.org/)


### Install

```
$ git clone https://github.com/recognai/biome.git
$ cd biome
$ make install
```

### Development

```
$ git clone https://github.com/recognai/biome.git
$ cd biome
$ make dev
```

If you do not want to install the packages required for testing, just use:

```
$ pip install -e .
```
