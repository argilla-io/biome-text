    biome.text.commands.serve.serve

    make_app

    make_app(binary: str, output: str = None)

    This function allows to serve a model from a gunicorn server. For example:

    gunicorn 'biome.allennlp.commands.serve.serve:make_app("/path/to/model.tar.gz")'

    Parameters

    binary Path to the model.tar.gz file output Path to the output folder, in which to store the predictions.

    Returns

    app A Flask app used by gunicorn server
